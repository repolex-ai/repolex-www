Tonight I'm at Frontier Tower in SF talking about **episodic memory for AI** — and how agents and humans can build a *shared visual memory* together.

Three months ago I gave a talk about [GitLex](/talks/circuit-launch-april-2026/) — an RDF knowledge graph attached to Git, a neutral substrate that turns any repo into a home for an agent. This is where it went.

The detour that became the project: I went to catch up on the generative space — diffusion, vision-to-text, SAM 3 segmentation, pose models — and the spark was, *what if the conversation itself fed the generation?* Each turn of a conversation produces a snapshot of whatever you're talking about. That turned into **Pool** ("Git for Images" — because images are like markdown: metadata on top, pixels as the body), a **chevron language** that lets agents author their own world mid-conversation, and **Punctum**, where you and the agent A/B-test generations to build a shared aesthetic over time.

The thread underneath it: this isn't memory *for* agents — it's memory *with* them. Collaborative taste and a visual narrative that compounds over months and years.
